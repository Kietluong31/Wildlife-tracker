Project Description

This Wildlife Sighting Tracker is an Express.js web application that loads wildlife observation records from a local JSON file and provides API endpoints to view and process the data.

The API demonstrates async data loading, JSON processing, and array methods such as filter(), map(), find(), and findIndex().

API Endpoints

Try these routes:

  • GET /api/sightings — all sightings
  • GET /api/sightings/verified — verified only
  • GET /api/sightings/species-list — unique species names
  • GET /api/sightings/habitat/forest — forest sightings + count
  • GET /api/sightings/search/eagle — first “eagle” match
  • GET /api/sightings/find-index/moose — index of Moose + record
  • GET /api/sightings/recent — 3 most recent sightings
